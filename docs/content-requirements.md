# Content Requirements

## Purpose

Define the minimum content needed before each major page type should be considered launch-ready.

## Home page

Required:
- hero heading
- hero subheading
- primary CTA
- secondary CTA
- company positioning section
- featured categories
- featured products
- featured manufacturers
- trust/strength section
- footer content
- meta title
- meta description

Recommended:
- short FAQ or support block

## About page

Required:
- page title
- core company description
- sourcing/trading capability section
- contact CTA
- meta title
- meta description

## Products index

Required:
- intro copy
- filter labels
- product cards with localized names
- pagination or listing behavior
- meta title
- meta description

## Category page

Required:
- localized category name
- localized slug path
- short intro
- body or explanatory block
- at least one published product in that locale
- meta title
- meta description

Recommended:
- child category links when relevant
- related manufacturers block
- inquiry CTA specific to the category

## Product detail page

Required:
- localized product name
- localized slug
- short description
- long description or equivalent structured body
- category
- manufacturer
- availability status
- availability note
- at least one image
- key specs where relevant
- inquiry CTA
- meta title
- meta description
- either:
  - a public price with currency
  - or a contact-for-pricing message

Recommended:
- downloadable product documents if available
- related products

## Manufacturers index

Required:
- intro copy
- manufacturer cards/list entries
- meta title
- meta description

## Manufacturer detail page

Required:
- localized manufacturer name
- localized slug
- summary
- body
- at least one related published product
- inquiry CTA
- meta title
- meta description

Recommended:
- logo or identifying image
- origin/country information

## Contact page

Required:
- localized intro
- inquiry form
- company contact details
- response expectation note
- privacy/consent text
- meta title
- meta description

## Locale requirements

For any locale to publish publicly:
- all required fields for that page/entity type must be filled
- the translation must be marked `published`
- the locale must be public

## Content quality bar

Content should be:
- specific
- business-relevant
- written for B2B readers
- not generic SEO filler
- not copied verbatim across categories/products/manufacturers without localization or differentiation
- clear about inquiry-led pricing when no public price is shown
