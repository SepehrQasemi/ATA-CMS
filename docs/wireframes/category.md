# Category Page Wireframe

## Page goal

Act as a strong SEO landing page for one category branch and route users into relevant product detail pages.

## Wireframe

### 1. Breadcrumbs

- Home
- Categories
- Parent category if applicable
- Current category

### 2. Category hero

- Category name
- Short category intro
- Optional supporting image

### 3. Child category links

- Show direct subcategories when the current category has children

### 4. Category overview block

- Deeper description of the category
- Optional bullet highlights or applications

### 5. Product listing

- Product cards or list rows for products in this category
- Show:
  - image
  - product name
  - manufacturer
  - availability
  - optional public price
  - CTA

### 6. Related manufacturers

- Manufacturer cards derived from products in this category

### 7. Inquiry CTA

- Prompt users to ask about sourcing this category

## Mobile notes

- hero copy comes before visuals
- product list stacks vertically
