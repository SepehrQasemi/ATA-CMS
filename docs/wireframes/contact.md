# Contact Page Wireframe

## Page goal

Make it easy for a qualified B2B visitor to contact ATA without implying online ordering.

## Wireframe

### 1. Breadcrumbs

- Home
- Contact

### 2. Page header

- H1: Contact / Inquiry
- Short explanation of response purpose and expected follow-up

### 3. Inquiry form

Fields:
- name
- company
- email
- phone
- country
- message
- optional product/manufacturer context when arriving from detail page
- consent checkbox

### 4. Direct contact block

- email
- phone
- office/address if appropriate

### 5. Response expectation block

- how ATA will respond
- what information helps qualify an inquiry

### 6. Optional contextual panel

- if user came from a product page:
  - show selected product summary
  - show price or contact-for-pricing state
- if user came from a manufacturer page:
  - show selected manufacturer summary

## Mobile notes

- keep form linear
- avoid multi-column layout on small screens
