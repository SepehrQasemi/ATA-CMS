# Manufacturers Index Wireframe

## Page goal

Expose ATA's manufacturer/producer network as a discovery and trust layer.

## Wireframe

### 1. Breadcrumbs

- Home
- Manufacturers

### 2. Page header

- H1: Manufacturers
- Intro paragraph

### 3. Manufacturer listing

- Manufacturer cards with:
  - logo or emblem
  - manufacturer name
  - short summary
  - origin country if available
  - related products CTA

### 4. Optional filter controls

- keyword search
- origin/country filter if content justifies it

### 5. Inquiry CTA

- ask ATA about products from a specific producer/brand

## Mobile notes

- cards become single-column
- keep summaries short and scan-friendly
