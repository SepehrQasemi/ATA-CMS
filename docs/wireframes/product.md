# Product Detail Page Wireframe

## Page goal

Give enough information for a B2B visitor to understand the product, its manufacturer, any public price, and how to inquire.

## Wireframe

### 1. Breadcrumbs

- Home
- Category hierarchy
- Product

### 2. Product hero

- Product name
- Category link
- Manufacturer link
- Availability badge
- Availability message
- Price block if public price exists
- Otherwise: contact-for-pricing message
- Primary CTA: Inquiry

### 3. Main content split

Left column:
- image gallery

Right column:
- short summary
- key facts/spec highlights
- CTA block

### 4. Detailed description

- long-form product description

### 5. Specifications table or cards

- spec label
- spec value

### 6. Manufacturer block

- manufacturer logo or name
- short manufacturer summary
- link to manufacturer page

### 7. Documents / downloads

- datasheet/catalog/certificate links when available

### 8. Related products

- cards or compact list

### 9. Inquiry section

- embedded form or strong CTA to contact page with product context

## Mobile notes

- image gallery stays above specs
- inquiry CTA repeated after description
