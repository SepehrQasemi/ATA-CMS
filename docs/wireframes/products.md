# Products Index Wireframe

## Page goal

Help users browse the product catalog efficiently without making the experience feel like an internal admin tool.

## Wireframe

### 1. Breadcrumbs

- Home
- Products

### 2. Page header

- H1: Products
- Intro paragraph

### 3. Filter / browse controls

- Keyword search
- Category filter
- Manufacturer filter
- Availability filter

### 4. Product results area

- Responsive card grid or hybrid list
- Each product card contains:
  - image
  - product name
  - category
  - manufacturer
  - availability status
  - optional public price
  - short summary
  - CTA to detail page

### 5. Optional featured categories strip

- Quick jump cards to category pages

### 6. Pagination

- Clear previous / next or numbered pagination

### 7. Bottom CTA

- Contact us for sourcing support

## Mobile notes

- filters collapse into compact controls
- cards remain readable without relying on hover
